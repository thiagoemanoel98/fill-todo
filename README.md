## Instalação

Clone o projeto e acesse a sua pasta:

```sh
$ git clone https://github.com/thiagoemanoel98/fill-todo.git
$ cd fill-todo
```

Instale as dependências do projeto e inicie:

```sh
# Dependências
$ yarn start

# Start
$ npx expo start

```
